这一部分完成了“展示诺贝尔奖得主学术产出的时间分布（获奖前后变化）”和“探索获奖论文发表年份与获奖年份的时间差”两个可视化目标。

“展示诺贝尔奖得主学术产出的时间分布（获奖前后变化）”部分使用折线图，纵轴为出版记录数量，横轴为年份，从诺奖得主第一次有出版记录的年份开始，到诺奖得主有出版记录的最后一年结束。
在折线图中，用红色虚线标注出了获奖年份，以此判别诺奖得主在获奖前后的学术产出变化；
支持用户根据获奖领域（chemistry，physics，medicine）和诺奖得主名字进行筛选；
用户可以将鼠标放在折线图上查看具体年份的具体出版数量。

“探索获奖论文发表年份与获奖年份的时间差”部分，由于有些诺奖得主的获奖论文不止一篇，因此在折线图中用不同颜色的虚线对其出版年份进行标注，最早的用黄色虚线标注。
时间差 = 获奖年份 - 使用获奖论文中最早出版的一篇的出版年份
然后在折线图下方，用橙色圆圈表示“获奖论文发表年份与获奖年份的时间差”，圆圈内部标注具体年份，圆圈大小与时间差成正比。
